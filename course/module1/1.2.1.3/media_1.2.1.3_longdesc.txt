La figura es una animación de un router recibiendo un paquete de datos.  La figura muestra que el router acepta el paquete porque la dirección MAC de destino es la del router.  El router lee la dirección IP de destino y mira su tabla de routing para determina adónde reenviar el paquete.  El router determina que el paquete debe enviarse a una interfaz Ethernet en el router siguiente.  El router encapsulará los datos, usará la interfaz del siguiente salto como la dirección MAC de destino y utilizará su propia dirección MAC como dirección de origen.  Las direcciones IP de origen y destino no se cambian.
