La figura muestra un switch con dos computadoras y un servidor FTP conectado a él.  Hay un enlace del switch a un router. El router se conecta a otro router, que se conecta a un tercer router.  El gateway predeterminado para las dos computadoras y el servidor es el puerto de router que está conectado al switch.   El tercer router tiene un switch conectado y un servidor web está conectado al switch.  La figura muestra una PDU que se forma cuando una de las computadoras necesita comunicarse con el servidor web.  La PDU consta de la dirección MAC de destino, que será la dirección MAC del gateway predeterminado, ya que el servidor web no se encuentra en la red local y el paquete se debe enviar al router.  La MAC de origen será la de la computadora que envía la solicitud y la dirección IP de origen será la de la computadora de envío.  La dirección IP de destino será la del servidor web.
