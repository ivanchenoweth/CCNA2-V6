La figura es una animación de un router recibiendo un paquete de datos.  La figura muestra que el router acepta el paquete porque la dirección MAC de destino es la del router.  El router lee la dirección IP de destino y mira su tabla de routing para determina adónde reenviar el paquete.  El router determina que debe enviarse un paquete por una interfaz serial, que no tiene una dirección MAC.  El router encapsulará los datos mediante una direcciónn de difusión para la información de capa dos.  Las direcciones IP de origen y destino no se cambian.
