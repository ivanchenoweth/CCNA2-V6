La figura uno muestra un ejemplo con comandos de filtrado con Cisco IOS.  En el ejemplo se ingresa el comando show running guión config barra vertical section line vty y el resultado es la sección de la configuración en ejecución que corresponde a las líneas de VTY. La figura dos muestra un ejemplo de uso de un comando de filtrado, y es show ip interface brief barra vertical include up.  En el resultado figuran tres interfaces como "up", lo que significa que están activas. La figura tres muestra un ejemplo de uso de un comando de filtrado, y es show ip interface brief barra vertical exclude unassigned.  En el resultado figuran tres interfaces como "up", lo que significa que están activas.  Es clara la ausencia de la cuarta interfaz en el router que no tiene asignada actualmente una dirección IP, por lo tanto esta dirección de interfaz no se asigna y se omite del resultado. La figura cuatro muestra un ejemplo de uso de un comando de filtrado, y es show ip route barra vertical begin Gateway.  El resultado muestra el contenido del comando show ip route comenzando por la palabra clave gateway y siguiendo hasta el final del resultado. La figura cinco es un elemento de medios interactivos que permite a los estudiantes practicar los comandos de filtrado.
