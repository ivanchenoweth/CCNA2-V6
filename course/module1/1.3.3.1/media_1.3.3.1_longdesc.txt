La figura muestra una red que consta de dos routers y cinco direcciones de red entre los dos.  En el primer router se configura una ruta predeterminada al segundo router.  Esto es así porque el siguiente router es el router de salto siguiente para todo el tráfico de red que proviene del primer router, incluida Internet.  Una ruta predeterminada es la configuración más eficiente para el router.  El segundo router sólo necesita alcanzar dos redes que están conectadas directamente al primer router.  Para alcanzar con mayor eficacia esas dos redes, se configuran dos rutas estáticas en el segundo router que apuntan al primer router.
