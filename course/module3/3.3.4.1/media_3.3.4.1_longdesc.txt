La figura muestra una topología de tres routers configurados en un diseño triangular. Cada router está conectado a los otros dos routers, lo que proporciona una topología de malla completa.  Cada router tiene dos enlaces WAN serie activos con direcciones IPv6 y cada router tiene una LAN en una interfaz de gigabit Ethernet con una dirección IPv6.  El protocolo de routing utilizado es EIGRP para IPv6.
