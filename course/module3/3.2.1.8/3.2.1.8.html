Packet Tracer: configuración de RIPv2

Si bien el protocolo RIP se utiliza con muy poca frecuencia en las redes modernas, es útil como base para comprender el routing de red básico. En esta actividad, configurará una ruta predeterminada, RIPv2, con las instrucciones de red e interfaces pasivas que correspondan, y verificará la conectividad total.
