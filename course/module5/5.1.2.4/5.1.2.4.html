Verificación de la configuración de puertos de un switch

En la figura 1, se describen algunas de las opciones del comando show que son útiles para verificar las características configurables comunes de un switch.

En la figura 2, se muestra un resultado abreviado del comando show running-config. Use este comando para verificar que el switch se haya configurado correctamente. Como se observa en el resultado de S1, se muestra cierta información importante:

  • Interfaz Fast Ethernet 0/18 configurada con la VLAN 99 de administración

  • VLAN 99 configurada con la dirección IPv4 172.17.99.11 255.255.255.0

  • Gateway predeterminado establecido en 172.17.99.1

El comando show interfaces es otro comando de uso frecuente que muestra información estadística y de estado sobre las interfaces de red del switch. El comando show interfaces se usa habitualmente cuando se configuran y se controlan los dispositivos de red.

En la figura 3, se muestra el resultado del comando show interfaces fastEthernet 0/18. En la primera línea de la ilustración, se indica que la interfaz FastEthernet 0/18 está “up/up”, lo que significa que está en funcionamiento. Más abajo en el resultado, se muestra que el modo dúplex es full (completo) y la velocidad es de 100 Mb/s.

La figura uno muestra la tabla que enumera los comandos show útiles para el uso en los switches. El primer comando enumerado es show interfaces, que se utiliza para mostrar el estado y la configuración de las interfaces. El siguiente comando enumerado es show startup guión config, que se utiliza para mostrar el archivo de configuración de inicio. El siguiente comando enumerado es show running guión configuration, que se utiliza para mostrar la configuración en ejecución. El siguiente comando enumerado es show flash, que mostrará el sistema de archivos flash. El siguiente comando enumerado es show version, que mostrará el estado del hardware y el software. El siguiente comando enumerado es show history, que mostrará el historial de los comandos introducidos. El siguiente comando enumerado es show ip interface guión id, que muestra la información de IP para la interfaz específica. El último comando enumerado es show mac guión address guión table, que mostrará la tabla de direcciones MAC V. La figura dos muestra el resultado del comando show running guión config. El resultado se abrevia para mostrar información sobre el puerto de switch cero barra 18, incluida la asignación de LAN virtual para esa interfaz. El resultado también muestra la dirección IP de la LAN 99 virtual que se configura en el switch y el gateway predeterminado. La figura tres muestra el resultado del comando show interfaces fast Ethernet cero barra 18. Este resultado proporciona detalles sobre la interfaz específica, incluida la configuración de velocidad y duplex, el funcionamiento actual y la dirección IP.
