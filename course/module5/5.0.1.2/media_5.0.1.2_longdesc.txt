La figura muestra dos conjuntos de switches. Un conjunto se denomina Comunicación por dúplex completo y semidúplex. Este conjunto está conectado con un cable Ethernet y posee la capacidad para enviar y recibir datos simultáneamente. El segundo conjunto de switches se denomina Comunicación por semidúplex. Este conjunto posee la capacidad para enviar o recibir datos.  Los switches utilizan la dirección MAC para identificar los hosts conectados, registran direcciones host únicas en una tabla de direcciones MAC y envían y reciben tráfico de datos mediante unidifusión, multidifusión y difusión.
