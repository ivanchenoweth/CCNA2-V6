La figura uno muestra el mensaje de descubrimiento de DHCPv4. Los mensajes consisten en la dirección MAC del cliente como el origen, una dirección MAC de destino de todas las F, una dirección IP 0.0.0.0 que se utiliza como IP de origen, la máscara de subred y el gateway predeterminado.  La dirección de destino es 255.255.255.255. La figura dos muestra el mensaje de oferta DHCPv4.  El servidor responde ofreciendo una dirección IP y una máscara de subred.
