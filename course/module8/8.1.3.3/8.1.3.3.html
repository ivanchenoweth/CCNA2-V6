Packet Tracer: configuración de DHCPv4 mediante el IOS de Cisco

Un servidor DHCP dedicado es escalable y relativamente fácil de administrar, pero existen situaciones donde no es rentable. Por ejemplo, en una sucursal pequeña o ubicación SOHO, se puede configurar un router Cisco para proporcionar servicios DHCPv4 sin necesidad de un servidor dedicado. El software Cisco IOS admite un servidor DHCPv4 con funciones completas opcional. El servidor DHCPv4 arrienda configuraciones por 24 horas de manera predeterminada. Como técnico de red de la empresa, tiene la tarea de configurar un router Cisco como servidor de DHCP para proporcionar la asignación dinámica de direcciones a los clientes de la red. También se le pide que configure el router perimetral como cliente DHCP para que reciba una dirección IP de la red ISP.
