La figura 1 muestra una topología de red de tres routers.  R2 se encuentra en el medio y tiene un enlace WAN serie con R1 y un enlace WAN serie con R3.  R1 y R3 no están directamente conectados.  R1 tiene dos redes LAN Gigabit Ethernet activas. Cada una de esas redes LAN tiene un switch conectado a ellas y cada switch está conectado a una computadora. También hay un servidor dhcp conectado a S2.  R3 tiene una LAN Gigabit Ethernet activa que contiene un switch y una computadora. La figura dos muestra los comandos utilizados para configurar un agente de retransmisión DHCPv6.  El ejemplo comienza en el modo de configuración global y pasa primero a la interfaz específica con el comando interface g 0 barra 0.  El siguiente comando es ipv6 dhcp relay destination 2 0 0 1 dos puntos d b 8 dos puntos c a f e dos puntos 1 dos puntos dos puntos 6.  En la figura se muestra el resultado del comando show ipv6 dhcp interface g 0 barra 0.  Este resultado indica que la interfaz está en modo de retransmisión y enumera la dirección IP de retransmisión configurada. La figura tres es una actividad interactiva que permite que el estudiante configure un agente de retransmisión DHCPv6.
