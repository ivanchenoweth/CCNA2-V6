La figura uno muestra el resultado del comando show ipv6 dhcp pool.  Esto muestra la configuración DHCPv6 en el router. La figura dos muestra el resultado del comando show ipv6 interface g 0 barra 1.  En la figura se resaltan las líneas que indican que el router tiene habilitada la configuración automática de dirección sin estado, la dirección IPv6 y la configuración de la dirección del router predeterminado. La figura tres muestra el resultado del comando debug ipv6 dhcp detail.  En la figura están resaltados todos los mensajes de DHCPv6 enviados entre el cliente y el servidor, como las opciones de nombre de dominio y del servidor DNS que se configuraron en el servidor. La figura 4 es una actividad interactiva que permite que el estudiante configure y verifique DHCPv6 en un router.
