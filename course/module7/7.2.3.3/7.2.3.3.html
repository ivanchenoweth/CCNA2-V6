Packet Tracer: configuración de una ACL IPv4 en líneas VTY

Como administrador de red, debe tener acceso remoto al router. Este acceso no debe estar disponible para otros usuarios de la red. Por lo tanto, configurará y aplicará una ACL que permita el acceso de una computadora (PC) a las líneas Telnet, pero que deniegue el resto de las direcciones IPv4 de origen.
