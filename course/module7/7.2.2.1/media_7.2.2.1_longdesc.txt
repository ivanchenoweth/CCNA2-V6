La figura muestra el proceso para usar un editor de texto para editar un A.C.L numerada.  El primer comando crea una A.C.L numerada con el número 1. La A.C.L se puede ver mediante el comando show running guión config pipe include access guión list 1.  La A.C.L puede copiarse y pegarse en un editor de texto y editarse.  La A.C.L actual se elimina de la interfaz con la palabra clave no. La nueva A.C.L se pega en el router y puede verificarse con el comando show running guión config pipe include access guión list 1.
