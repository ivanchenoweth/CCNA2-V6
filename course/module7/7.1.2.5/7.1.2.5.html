Ejemplos de palabras claves de máscara wildcard

En el ejemplo 1 en la ilustración, se muestra cómo utilizar la palabra clave any para sustituir la dirección IPv4 0.0.0.0 por una máscara wildcard 255.255.255.255.

En el ejemplo 2, se muestra cómo utilizar la palabra clave host para sustituir la máscara wildcard para identificar un único host.

Nota: La sintaxis para configurar ACL de IPv4 estándar se explica más adelante en este capítulo.

La figura muestra dos ejemplos de uso de los comandos de C.L.I para crear una lista de acceso que permita cualquier dirección. Los ejemplos comienzan en el modo de configuración global. El próximo ejemplo utiliza la palabra clave, y el comando es access guión list 1 permit any. El segundo ejemplo es un A.C.L que especifica un host determinado.
