Estas figuras se basan en el diagrama de laboratorio estándar.  R1 con una LAN se conecta a R2 con una LAN que se conecta a R3 con una LAN. La figura 1 muestra la configuración de las dos LAN a las que el router 1 no está conectado, la LAN del router 2 y la LAN del router 3, además de la WAN entre el router 2 y el router 3.  Las rutas estáticas se configuraron con una ruta estática totalmente calificada.  Este tipo de ruta tiene interfaz de salida y dirección de siguiente salto. La figura 2 muestra el proceso de búsqueda para que el router 1 se comunique con la LAN en el router 3.  La figura tres es una actividad interactiva que permite que el estudiante configure una ruta estática totalmente calificada en el router dos para llegar a la LAN del router tres. La figura 4 es una actividad interactiva que permite que el estudiante configure rutas estáticas en el router 3 a las redes del otro lado del router 2.
