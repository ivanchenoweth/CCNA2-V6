Estas figuras se basan en el diagrama de laboratorio estándar.  R1 con una LAN se conecta a R2 con una LAN que se conecta a R3 con una LAN.  Las figuras uno a tres muestran las tablas de routing de cada router.  Cada router enumera solo las redes conectadas directamente en la tabla de routing. La figura cuatro muestra el router uno haciendo ping al otro extremo del enlace serial, al router dos.  Este ping tiene éxito porque son routers conectados directamente. La figura cinco muestra el router uno intentando hacer ping a la interfaz LAN del router tres.  Este ping no es exitoso ya que los routers solo conocen sus redes conectadas directamente.  El router uno no tiene ninguna ruta a la LAN o el router tres.
