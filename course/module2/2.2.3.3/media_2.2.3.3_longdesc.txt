Estas figuras se basan en el diagrama de laboratorio estándar.  R1 con una LAN se conecta a R2 con una LAN que se conecta a R3 con una LAN. La figura 1 muestra la configuración de las rutas estáticas a las redes remotas en el router 1 usando IPv6.  Las rutas estáticas se configuraron usando la dirección del próximo salto. La figura 2 muestra el proceso de búsqueda recursiva para que el router 1 se comunique con la LAN en el router 3.  Este proceso se requiere siempre que se ingrese una ruta estática usando un próximo salto en lugar de una interfaz de salida. La figura tres es una actividad interactiva que permite que el estudiante configure una ruta estática en el router dos, mediante IPv6, para llegar a la LAN del router tres utilizando la dirección de siguiente salto. La figura 4 es una actividad interactiva que permite que el estudiante configure rutas estáticas en el router 3 usando IPv6, apuntando a las redes del otro lado del router 2.
