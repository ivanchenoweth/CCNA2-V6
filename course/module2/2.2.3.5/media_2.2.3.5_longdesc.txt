Estas figuras muestran dos routers conectados con un cable serial.  Cada router tiene conectada una LAN. En la figura uno se muestra la sintaxis para configurar una ruta estática totalmente calificada en el router uno a la LAN en el router dos.  Se requiere la instrucción de ruta totalmente calificada porque estamos utilizando la dirección de enlace local como siguiente salto.  Esto requiere el uso de una interfaz de salida. La figura dos muestra el resultado de emitir el comando “show ipv6 route static barra vertical begin followed by an ipv6 address” para el router. La figura tres es una actividad interactiva que permite a los estudiantes practicar la configuración de una ruta estática totalmente calificada en IPv6.
